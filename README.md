Welcome to **Hearts Card Game**! A classic, fun, and competitive trick-taking card game beautifully designed for Android devices.

## 📱 About the Project

Hearts is an engaging trick-taking card game where the objective is to have the lowest score at the end of the game. Players must strategically avoid winning tricks containing penalty cards (Hearts and the Queen of Spades) while attempting to shoot the moon if they feel daring!

This application provides a modern and smooth mobile experience for playing Hearts against intelligent AI opponents.

## ✨ Features

- **Classic Gameplay:** Standard Hearts rules with intuitive touch controls.
- **Smart AI Opponents:** Play against challenging computer players.
- **Sleek UI/UX:** Clean, modern, and engaging user interface.
- **Score Tracking:** Keeps track of points across multiple rounds.
- **Offline Play:** Enjoy the game anytime, anywhere without an internet connection.

## 🛠️ Tech Stack

- **Platform:** Android
- **Language:** Kotlin
- **Build System:** Gradle (Kotlin DSL)
- **Architecture:** MVVM (Model-View-ViewModel) / Standard Android Architecture components

## 🚀 Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing.

### Prerequisites

- [Android Studio](https://developer.android.com/studio) (Latest Version Recommended)
- Minimum Android SDK required: API 24 (or as specified in `build.gradle.kts`)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/abidali72/Heart-Card-Game.git
   ```
2. **Open the project in Android Studio**
   - Launch Android Studio.
   - Select `Open an Existing Project`.
   - Navigate to the cloned directory and select it.

3. **Sync Gradle**
   - Wait for Android Studio to index the files and sync the Gradle project.

4. **Run the App**
   - Select your emulator or connected Android device.
   - Click the **Run** button (green play icon) in the toolbar.
